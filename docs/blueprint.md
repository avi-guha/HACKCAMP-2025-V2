# **App Name**: ToneScribe

## Core Features:

- Screenshot Upload: Allow users to upload screenshots of text message conversations.
- AI-Powered Text Extraction: Utilize AI to accurately extract text from the uploaded screenshot.
- Line-by-Line Tone Decoding: Employ AI to decode the tone of each message line (e.g., positive, negative, neutral, sarcastic). The AI should use a 'tool' to decide the decoding. 
- Tone Visualization: Display the tone of each message using distinct colors and/or icons for easy understanding.
- Sentiment Summary: Provide an overall sentiment summary of the entire conversation, highlighting key emotional shifts.
- Conversation Archiving: Let users archive screenshots and sentiment summaries to Firestore.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to give a calm, tech-forward impression.
- Background color: Off-white (#F0F0F0) to provide a clean and neutral base.
- Accent color: Pale violet (#BDB2FF) to complement the primary with a touch of technological novelty, while still providing sufficient contrast.
- Body and headline font: 'Inter', a sans-serif font, for a modern, objective, and neutral feel.
- Utilize simple, clear icons to represent different tone categories.
- Maintain a clean, whitespace-abundant layout to enhance readability.
- Incorporate subtle animations on sentiment analysis for enhanced engagement.
